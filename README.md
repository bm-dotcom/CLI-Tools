A personal showcase of **command-line interface (CLI) tools** built to explore.

This repository contains multiple independent CLI projects, each focused on different aspects of terminal tooling:

- **Terminal output** — rich formatting, tables, progress bars, spinners, and live updates  
- **Argument parsing & subcommands** — using industry-standard libraries (Typer, clap, argparse)  
- **Configuration & state management** — YAML/JSON/TOML support, env vars, persistent settings  
- **Low-level systems programming** — binary/file inspection, memory-efficient processing, entropy & stats in C  
- **Piping & composability** — clean JSON output, stdin/stdout handling, exit codes & error handling  
- **Packaging & distribution** — installable via pip/cargo/brew, single binaries where possible
